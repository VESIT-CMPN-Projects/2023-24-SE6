This website was developed as part of Project 2 of the Code Academy Berlin program. The project consisted to pull and display live data from an online API.

VanyaSparsh's API converts operational activity data into CO2e (carbon dioxide equivalent) estimates, allowing to calculate each activity's impact on the environment.

What is an Emission Factor? An emission factor is a value that estimates the quantity of a gas with a global warming potential (GWP) released into the atmosphere, per some unit related to a particular activity.
